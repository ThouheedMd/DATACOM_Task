## Azure App Registration & GitHub Actions Setup

## Project Overview
This project demonstrates the setup of essential DevOps tools and cloud services, along with creating an application registration in Microsoft Azure and implementing a basic GitHub Actions workflow.

The goal of this task was to gain hands-on experience with:
- Cloud identity management (Azure Active Directory)
- Development environment setup
- Version control using GitHub
- Basic CI/CD automation using GitHub Actions

---

## 🛠️ Tools & Technologies Used
- Microsoft Azure Portal
- Azure Active Directory (AAD)
- Git & GitHub
- GitHub Actions
- Visual Studio / VS Code

---

## ⚙️ Tasks Performed

### 1️⃣ Development Environment Setup
- Installed and configured Git
- Set up Visual Studio / VS Code
- Configured GitHub account

---

### 2️⃣ Azure Account Setup
- Created a Microsoft Azure Free Tier account
- Selected free subscription plan
- Verified account using credit card (no charges applied)

---

### 3️⃣ Microsoft 365 Developer Program
- Joined Microsoft 365 Developer Program
- Activated E5 Developer Subscription
- Explored developer sandbox environment

---

### 4️⃣ Azure Active Directory - App Registration
- Navigated to Azure Active Directory
- Created a new App Registration
- Configured:
  - Application Name
  - Supported Account Types
  - Redirect URI (if applicable)
- Generated Application (Client) ID and Directory (Tenant) ID

---

### 5️⃣ GitHub Repository Setup
- Created a new GitHub repository
- Cloned repository to local machine
- Managed files using Git commands

---

### 6️⃣ GitHub Actions Workflow
- Created a basic CI workflow using GitHub Actions
- Configured workflow to trigger on push events
- Executed a sample automation script
